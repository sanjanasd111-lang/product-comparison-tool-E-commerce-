## 🚀 How to Run the Project

### 🖥️ 1. Launching the Web Application
The web app serves as your central command center for all saved comparisons.

1.  *Clone the Repository:*
    bash
    git clone [https://github.com/YourUsername/Product-Comparison-Tool.git](https://github.com/YourUsername/Product-Comparison-Tool.git)
    cd Product-Comparison-Tool
   
2.  *Run the Server:*
    Locate the app.py file in the root directory and simply execute:
    bash
    python app.py
   
3.  *Access Localhost:*
    Once the server starts, open your browser and go to:
    http://127.0.0.1:5000 💻 (or the link shown in your terminal).

---

### 🧩 2. Loading the Chrome Extension
Once the local server is running, you can load the extension to your browser:

1.  Open *Google Chrome* and go to: chrome://extensions/
2.  Enable *Developer Mode* in the top right corner 🛠️.
3.  Click the *"Load unpacked"* button.
4.  Select the folder containing the extension files from this repository.
5.  The extension is now ready! You can see the Codesky icon in your toolbar. ✨
